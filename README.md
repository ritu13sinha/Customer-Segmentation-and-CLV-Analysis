# Customer Segmentation & Customer Lifetime Value (CLV) Analysis

## Overview

This project focuses on Customer Segmentation and Customer Lifetime Value (CLV) Analysis to identify valuable customer groups and support data-driven business decisions. The project includes data preprocessing, exploratory data analysis, customer segmentation, and an interactive Power BI dashboard.

## Project Objectives

- Analyze customer purchasing behavior.
- Segment customers based on their characteristics.
- Calculate and evaluate Customer Lifetime Value (CLV).
- Identify high-value customers.
- Provide business recommendations using data insights.

## Project Workflow

- Phase 1 – Data Cleaning & Preprocessing
- Phase 2 – Exploratory Data Analysis (EDA)
- Phase 3 – Customer Segmentation & CLV Analysis
- Phase 4 – Interactive Power BI Dashboard

## Key Business Insights

- VIP customers contribute the highest Customer Lifetime Value.
- Loyal customers generate consistent revenue through repeat purchases.
- At-Risk customers can be targeted with retention and re-engagement campaigns.
- Customer segmentation helps businesses prioritize marketing strategies and improve customer retention.

## Project Outcome

This project demonstrates an end-to-end data analytics workflow, from data preparation and analysis to business insights and dashboard development using Power BI.
